# Logseq graph manifest.
#
# One line per graph:   <FolderName> <branch>
#   FolderName  the directory the graph lives in (a git worktree), opened in Logseq
#   branch      the branch that holds that graph's content (convention: graphs/<FolderName>)
#
# This file is committed on `main`, so every device knows which graphs exist and
# how to reconstruct them. Edit it via ./add-graph.sh (don't hand-edit unless you
# know what you're doing). Blank lines and lines starting with # are ignored.
#
# Example:
# MyGraphA graphs/MyGraphA
# MyGraphB graphs/MyGraphB
